
**Tools to Install:**

1. **In VS Code:**
    
    - Install the **Live Sass Compiler** plugin by **Ritwick Dey**.
2. **In Visual Studio:**
    
    - Go to **Manage Extensions**, search for **"Web Compiler"** by **Jason Moore**, and install it.
    - After installation, right-click on any SCSS file and select **Web Compiler > Compile** to compile the SASS.
