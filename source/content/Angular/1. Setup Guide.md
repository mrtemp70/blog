
## Typescript download

1. First, download and install Node.js. (Must install into C Drive)
2. To check the installation, use the following commands:
   - `node -v`
   - `npm -v` (This is a library of node. It download auto with node installation but if need latest version `npm install -g npm@latest`)
4. To check the TypeScript version: `tsc -v` (install TypeScript globally `npm install -g typescript)
5. Download the ESLint extension in VS Code.


## Angular Download

1. First, download and install Node.js (Or, `nvm install 22.13.1`).
2. Install Angular CLI globally using the command:  
   `npm install -g @angular/cli`
3. To check the Angular version, use the command:  
   `ng v`  
   *(Press `y` when prompted for the first time.)*
