

1. Create new project `ng new project-name --standalone=false` (Here NgLab kind of project-name. AppModule will not be visible if standalone=true)
2. To run project `ng serve --o`
3. To create a angular component use `ng g c component-name --standalone true --skip-tests` 
4. To create a module and routing (both in one command): `ng g m module-name --routing`
